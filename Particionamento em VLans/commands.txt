Start module of kernel 802.1Q -> sudo modprobe 8021q
Create a vlan.10 (ADM) -> sudo ip link add link enp0s3 name enp0s3.10 type vlan id 10
Create a vlan.20 (Web) -> sudo ip link add link enp0s3 name enp0s3.20 type vlan id 20
Create a vlan.30 (Vunerable Machines) -> sudo ip link add link enp0s3 name enp0s3.30 type vlan id 30
Up vlan 10 -> sudo ip link set enp0s3.10 up
Up vlan 20 -> sudo ip link set enp0s3.20 up
Up vlan30 -> sudo ip link set enp0s3.30 up
